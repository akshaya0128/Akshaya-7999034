📚 Help Centre

Answers to common questions about the CityConnect Community Portal.

How do I register for an event?

Navigate to the Register section, fill in your name, email, preferred date, and event type, then click Register Now. You will see an on-screen confirmation and receive an email shortly after.

Can I save my event preferences?

Yes! Visit the Save Your Preferences section, choose your favourite event type, and it will be remembered for your next visit using localStorage. Click Clear All Preferences to remove saved data.

How does Find Nearby Events work?

The portal uses your browser's Geolocation API (with high accuracy mode). When you click Find Nearby Events, your browser will ask for permission. If you allow it, we display your coordinates and a map link showing events within 10 km.

Why do I see a warning when leaving the page?

If you have started filling out a form but have not submitted it, the portal uses the onbeforeunload event to warn you so you don't accidentally lose your data.

What is the Debug Console?

The Debug Console at the bottom of the page logs all user interactions in real time. It mirrors output to the browser's built-in Console tab (F12 → Console), making it useful for learning HTML5 event handling alongside Chrome DevTools.
