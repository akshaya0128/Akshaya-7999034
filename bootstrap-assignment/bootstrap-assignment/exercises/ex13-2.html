How it works

Bootstrap 5 applies a gradient by adding a CSS background-image on top of the background-color:

.bg-gradient {
  background-image: var(--bs-gradient);
}
/* --bs-gradient is defined as: */
--bs-gradient: linear-gradient(180deg, rgba(255,255,255,.15), rgba(255,255,255,0));

Note: $enable-gradients: true in Sass applies gradients globally to all components. Without it, you must add .bg-gradient manually.
