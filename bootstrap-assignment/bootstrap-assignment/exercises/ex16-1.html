Exercise 16.1 – Fixed Footer

The footer below is fixed at the bottom of the viewport using position-fixed bottom-0 start-0 end-0. It stays visible as you scroll.

Scroll down to see the fixed footer always stay at the bottom.

Position Utilities Used
  • position-fixed — removes the element from normal flow and fixes it relative to the viewport
  • bottom-0 — positions it at the bottom (bottom: 0)
  • start-0 — stretches to the left edge (left: 0)
  • end-0 — stretches to the right edge (right: 0)
Other fixed positioning examples
  • position-fixed top-0 — sticky top navbar
  • position-sticky top-0 — sticky (scrolls until edge, then sticks)
  • position-fixed bottom-0 end-0 — floating action button (FAB)
