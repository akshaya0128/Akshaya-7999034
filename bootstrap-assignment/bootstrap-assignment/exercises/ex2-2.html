Exercise 2.2 – Including bootstrap.bundle.min.js

bootstrap.bundle.min.js includes Bootstrap's own JS plugins plus Popper.js, so you only need one <script> tag.

Why the bundle? Several Bootstrap components (dropdowns, tooltips, popovers) rely on Popper.js for positioning. The bundle ships both together so you don't manage two separate dependencies.
Where to place the script tag
<!-- Place just before </body> for best performance -->
<script src="js/bootstrap.bundle.min.js"></script>

<!-- Or via CDN -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
