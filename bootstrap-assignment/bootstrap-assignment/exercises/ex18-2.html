2. Flush Accordion (accordion-flush, multiple open)

Without data-bs-parent, multiple panels can be open simultaneously. This is the "always open" mode.

The accordion-flush class removes the outer border and border radius for a flat, edge-to-edge look inside a container.

This panel can be opened while the others remain open — no data-bs-parent means no mutual exclusion.
