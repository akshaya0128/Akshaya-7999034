Exercise 1.2 – Downloaded Bootstrap Setup

Demonstrates how to wire up Bootstrap using locally downloaded files.

Local File Reference Guide

After downloading and extracting Bootstrap, your project structure should look like:

project/
├── css/
│   └── bootstrap.min.css        ← compiled Bootstrap styles
├── js/
│   └── bootstrap.bundle.min.js  ← Bootstrap + Popper bundled
└── index.html                   ← your HTML page

Then in your HTML:

<!-- In <head> -->
<link rel="stylesheet" href="css/bootstrap.min.css" />

<!-- Before </body> -->
<script src="js/bootstrap.bundle.min.js"></script>
Bootstrap is loaded and styles are applied correctly.
