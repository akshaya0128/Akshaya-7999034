Exercise 19.1 – Bootstrap 5 with Sass Setup

This page documents the steps to set up Bootstrap 5 with Sass using npm. The compiled CSS for this demo still uses the CDN; the steps below explain the local workflow.

Step 1 – Initialise a Node project
mkdir my-bootstrap-project
cd my-bootstrap-project
npm init -y
Step 2 – Install Bootstrap and Sass
npm install bootstrap
npm install --save-dev sass

This installs Bootstrap's Sass source files under node_modules/bootstrap/scss/.

Step 3 – Create your custom Sass entry point

Create scss/custom.scss:

// scss/custom.scss

// 1. Override Bootstrap variables BEFORE importing Bootstrap
$primary: #6f42c1;        // purple instead of blue
$font-size-base: 1rem;
$border-radius: 0.5rem;

// 2. Import Bootstrap source
@import "../node_modules/bootstrap/scss/bootstrap";
Step 4 – Compile Sass to CSS
# One-time compile
npx sass scss/custom.scss css/custom.css

# Watch mode (recompile on every save)
npx sass --watch scss/custom.scss css/custom.css
Step 5 – Link compiled CSS in HTML
<!-- Replace the Bootstrap CDN link with your compiled file -->
<link rel="stylesheet" href="css/custom.css" />

<!-- JS still from CDN or node_modules -->
<script src="node_modules/bootstrap/dist/js/bootstrap.bundle.min.js"></script>
Recommended Project Structure
my-bootstrap-project/
├── node_modules/          ← npm packages
├── scss/
│   ├── custom.scss        ← your entry point (imports bootstrap)
│   └── _variables.scss    ← optional: move overrides here
├── css/
│   └── custom.css         ← compiled output (gitignored)
├── js/
│   └── main.js
├── index.html
└── package.json
Optional: Add an npm script in package.json
"scripts": {
  "sass": "sass --watch scss/custom.scss css/custom.css"
}

Then run with npm run sass instead of the long npx command.
