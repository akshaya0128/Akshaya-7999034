Exercise 19.2 – Customise with _variables.scss

scss/_variables.scss

Override Bootstrap variables before the Bootstrap import:

// ─── Colours ───────────────────────────
$primary:        #6f42c1;   // purple
$secondary:      #fd7e14;   // orange
$success:        #20c997;   // teal
$danger:         #e63946;
$warning:        #ffd60a;

// ─── Typography ────────────────────────
$font-family-sans-serif: 'Inter', sans-serif;
$font-size-base:  1rem;      // 16px
$font-size-lg:    1.125rem;
$font-weight-bold: 700;
$headings-font-weight: 800;

// ─── Spacing ───────────────────────────
$spacer: 1rem;  // all spacing scales from this

// ─── Border radius ─────────────────────
$border-radius:       0.75rem;
$border-radius-sm:    0.375rem;
$border-radius-lg:    1rem;
$border-radius-pill:  50rem;

// ─── Breakpoints ───────────────────────
$grid-breakpoints: (
  xs: 0,
  sm: 576px,
  md: 768px,
  lg: 992px,
  xl: 1200px,
  xxl: 1400px
);

// ─── Shadows ───────────────────────────
$enable-shadows:   true;
$enable-gradients: true;
scss/custom.scss (import order)

Import only the Bootstrap modules you need (tree-shaking), or import everything:

// Option A: import all of Bootstrap
@import "../node_modules/bootstrap/scss/bootstrap";

// ─────────────────────────────────────────
// Option B: import selectively (smaller CSS)
// ─────────────────────────────────────────

// Required: functions, variables, mixins
@import "../node_modules/bootstrap/scss/functions";
@import "../node_modules/bootstrap/scss/variables";
@import "../node_modules/bootstrap/scss/variables-dark";
@import "../node_modules/bootstrap/scss/maps";
@import "../node_modules/bootstrap/scss/mixins";
@import "../node_modules/bootstrap/scss/root";

// ① Your overrides go BEFORE variables import
// ② Then add only the components you use:
@import "../node_modules/bootstrap/scss/reboot";
@import "../node_modules/bootstrap/scss/type";
@import "../node_modules/bootstrap/scss/grid";
@import "../node_modules/bootstrap/scss/buttons";
@import "../node_modules/bootstrap/scss/card";
@import "../node_modules/bootstrap/scss/modal";
@import "../node_modules/bootstrap/scss/utilities";
@import "../node_modules/bootstrap/scss/utilities/api";

Key Rules for Sass Customisation
  • Always override variables before importing Bootstrap Sass files.
  • Use !default in your own _variables.scss to allow further overriding.
  • Recompile after every change: npx sass --watch scss/custom.scss css/custom.css
  • Commit scss/ to git; ignore the compiled css/ output if preferred.
