Exercise 6.1 – Flex Navbar

The navbar above uses these flexbox utilities:

  • d-flex – enables flexbox on the nav element
  • flex-column – stacks items vertically on mobile
  • flex-md-row – switches to horizontal layout on md+ screens
  • align-items-start / align-items-md-center – cross-axis alignment per breakpoint
  • justify-content-md-between – spreads brand, links, and button apart on desktop
Resize the window below 768px to see the navbar collapse into a vertical stack.
