Exercise 11.2 – Media Object Layout

Bootstrap 5 replaces the old .media component with flexbox utilities. Use d-flex + flex-shrink-0 on the image and ms-3 on the body.

This is the media body. The image sits to the left, content flows to the right. Bootstrap 5 achieves this with d-flex on the wrapper and flex-shrink-0 on the image so it doesn't shrink when the text grows.
